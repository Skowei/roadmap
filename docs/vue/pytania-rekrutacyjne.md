<!-- 1. Opisz lifecycle hooki
```
hooks to funkcje, przez które przechodzi każda instancja Vue. Wyróżniamy ich 8.

- Before Create - jest to pierwszy hook, który jest wywoływany po utworzeniu instancji Vue.
- Created - jest wywoływany natychmiast po haku "Before create", ale instancja Vue ma już ustawione początkowe właściwości, dane itp.
- Before mount - wywoływany tuż przed zamontowaniem instancji w DOM. W tym momencie template został utworzony.
- Mounted - jest to nazwa nadawana szablonowi po wypełnieniu go danymi i uzyskaniu pełnej funkcjonalności.
- Before Updated - gdy jakiekolwiek zmiany danych wymagają zmiany DOM, ta metoda jest wywoływana przed aktualizacją.
- Updated - jest wywoływana po aktualizacji DOM.
- Before destroy - Jest to miejsce, w którym można wyczyścić zasoby przed zakończeniem instancji Vue.
- Destroyed - ta metoda jest wywoływana, gdy wszystkie instancje Vue zostaną zniszczone. Po wywołaniu metody destrukcji na obiekcie w kodzie, zostanie on aktywowany.
```
